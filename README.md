Асинхронный TCP-сервер для интеграции лабораторного оборудования.

Проект предназначен для приема, обработки и передачи данных от внешних устройств с возможностью дальнейшего расширения поддержки различных лабораторных протоколов.

## Поддерживаемое оборудование

На текущем этапе проект развивается в направлении интеграции со следующим оборудованием:

- Urit 5160Vet
- Accenty M320
- Seamaty SMT-120VP

## О проекте

Happy Laba — это open source проект, целью которого является создание универсального слоя интеграции между лабораторным оборудованием и информационными системами.

Проект развивается поэтапно: от базовой серверной инфраструктуры к полноценной платформе для обработки лабораторных данных, поддержки различных протоколов и дальнейшего расширения функциональности.

Если проект показался вам интересным, вы можете следить за его развитием через репозиторий, создавать Issues, предлагать улучшения или участвовать в разработке.

# Релиз 0.0.3
## Текущий статус

## Добавлено

- Добавлена поддержка нескольких TCP-каналов.
- Добавлен Runtime для управления жизненным циклом приложения.
- Добавлена поддержка конфигурации нескольких устройств через YAML.
- Добавлена проверка доступности TCP-портов.
- Добавлена проверка уникальности портов.
- Добавлен SessionContext для хранения состояния активных сессий.

## Изменено

- TCPServer переименован в TCPChannel.
- Переработана архитектура приложения для поддержки нескольких каналов.
- Переработан процесс инициализации приложения.
- Обновлен механизм управления сессиями с учетом типа канала.
- Унифицировано логирование.
- Переработана структура конфигурации приложения.
- Выполнен рефакторинг компонентов и улучшено разделение ответственности.

## Исправлено

- Исправлено завершение TCP-каналов.
- Исправлено завершение сессий по типу канала.
- Исправлены ошибки инициализации приложения.
- Исправлена обработка отмены фоновых задач.
- Исправлены ошибки валидации конфигурации.

## Тестирование

- Обновлены модульные тесты.
- Тесты адаптированы под многоканальную архитектуру.
- Исправлены тесты SessionManager.
